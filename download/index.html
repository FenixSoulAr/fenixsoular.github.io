Install as an app (PWA)

Recommended. Works without Play Store / App Store and feels like a native app.

Android (Chrome)

  1. Open https://salud.fenixsoular.com.ar/ in Chrome.
  2. Tap ⋮ (menu).
  3. Select Install app / Add to Home screen.

iPhone / iPad (Safari)

  1. Open https://salud.fenixsoular.com.ar/ in Safari.
  2. Tap Share.
  3. Select Add to Home Screen.
